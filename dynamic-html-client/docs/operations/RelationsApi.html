This is the API

relationsAncestorsGet

The list of ancestors.

URL

https://api.wuber.com/v1/relations/ancestors

HTTP Method

GET

Response Type

List

Parameters

  • Query: person String

    the name of the person.

  • Query: generation String

    the number of generation of ancestors.

  • Query: bloodrelative String

    Type of blood relative, lineal, collateral, adopted or all
