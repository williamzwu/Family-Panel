familyGet Family Tree around this family
The family endpoint returns information describing the family tree around specified family.

Parameters

familyname (required)
Query Parameter — The name of the family.
generation (required)
Query Parameter — The number of generation below the father of the asked family.

Return type

array[Family]

Example data

Content-Type: application/json
[ {
  "residence" : "aeiou",
  "family_name" : "aeiou"
} ]
